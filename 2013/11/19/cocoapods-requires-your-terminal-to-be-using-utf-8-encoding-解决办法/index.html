CocoaPods requires your terminal to be using UTF-8 encoding. 解决办法

cocoapod 升级到0.28.0 后会提示 “CocoaPods requires your terminal to be using UTF-8 encoding” 的警告，issue26中给出了解决办法，就是设置export LC_ALL=”en_US.UTF-8”，可以把这句话添加到.zshrc或者.bashrc中，然后source .zshrc或者source .bashrc就可以了。
