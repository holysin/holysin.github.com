XVim介绍

Aug 01, 2014

XVim是Xcode下的vim插件，而且支持vim的大部分功能，喜欢vim的同学，绝对不能错过这个插件。
下面简要介绍一下XVim的基本功能。

移动

基本

h,j,k,l 左，上，下，右

移动词

操作 作用
w 移到下一个词的开头
W 移到下一个字符串的开头
b 移到上一个词的开头
B 移到上一个字符串的开头
e 移到下一个词的末尾
E 移到下一个字符串的末尾
f {char} 向右找到char
F {char} 向左找到char
t {char} 向左找到char，光标停在char左边
T {char} 向右找到char，光标停在char右边
; 重复最近的f, t, F, T，可重复
, 重复最近的f, t, F, T，可重复，但是相反方向

行内移动

操作 作用
0 移到行首
^ 移到第一个非空字符
$ 移到行尾

段落移动

操作 作用
{ 移到上个段落开头
} 移到下个段落开头

页面移动

操作 作用
ctrl-f 向下移动一页
ctrl-b 向上移动一页
ctrl-d 向下移动半页
ctrl-u 向上移动半页

文件内移动

操作 作用
gg 移到文件头
G 移到文件末尾

插入

操作 作用
a 光标后插入
A 行后插入
i 光标处插入
I 行前插入
o 在光标所在行下面新起一行，插入
O 在光标所在行上面新起一行，插入

删除

操作 作用
x 删除光标处字符
X 删除光标前字符
dd 删除当前行
D 从光标删除至行尾

注意在visual 模式下，这些删除操作同样可以使用

修改 和 替换

操作 作用
r{char} 将光标处字符替换成char 注释： {}不需要输入
~ 改变当前字符的大小写
g~{移动} 改变移动范围内文字的大小写 注释： {}不需要输入
{visual}~ 可视模式改变选中区域的大小写 注释： visual 模式下

替换

:s/{pattern}/{string}/
:%s/{pattern}/{string}/g
在文件范围内，用string替换pattern。%表示所有行中，g替换行中所有匹配点。
:{visual}/{pattern}/{string} visual 模式下替换

跳转 (Xcode 特有)

操作 作用
ctrl-o, ctrl-i, gd 跳到函数定义
gd 当光标在某个文件处时，跳到该文件

行连接

J 注释：大写

排版

操作 作用
<< 左移一个tab
>> 右移一个tab
= 自动排版

文件内全局排版
gg=G

粘贴

操作 作用
y{count}y 复制count行
y{count}w 复制count词
p 光标后（下）粘贴
P 光标前（上）粘贴

用d操作删除的内容同样可以用p粘贴

修改

操作 作用
cw 删除光标所在词，并插入
c^ 删除到行前第一个字符，并插入
c0 删除到行前并插入
c$ 删除到行尾并插入

标记

操作 作用
m{char} 标记光标处的char
`{char} 回到标记处
:marks 列出所有标记

查找

操作 作用
/{pattern} 查找pattern
n / N 向下/上查找

可视模式

操作 作用
ctrl-v 纵向可视模式
shift-v 横向可视模式

. 操作符

重复上次操作
比如dd之后，按.就会重复dd。不能重复移动操作

常用技巧

1.在行后插入字符
:s/$/{char}/g
2.在行前插入字符
:s/^/{char}/g
3.列模式可视模式下，插入
ctrl-v选中列模式，shift-i插入字符，esc 3次
