群辉入门教程

Jul 31, 2014

先上图片，来自官网，我觉得机子还是很漂亮的，白色机身，非常干净。



安装步骤

安装步骤很简单，把外壳打开，把硬盘塞进去，盖上外壳就行了。注意包装里面有螺丝，是用来固定硬盘用的，不要忘了。
安装完毕后，用网线接上你的路由器，开启群辉的电源，这时候就可以在局域网中看到了。

软件设置

1.在浏览器中输入http://find.synology.com/，就可以打开联机群辉的页面了。

2.然后会出现登录界面，输入你的用户名，密码就进入到群辉的控制台了（暂且叫控制台吧）。


3.首先进入套件中心，下载必须的套件


群辉提供了老多老多的套件，先介绍几个简单常用的。

Download station

Download station和迅雷的界面差不多，大家一用应该就明白了，有一个url批量下载功能，你只需要把要下载的url放入一个文件，然后打开这个文件，群辉就会全部帮你下载。

File station

File station是用来管理群会上的文件的，操作也很简单，复制，粘贴，剪切，就是这些基本操作。介绍一下如何建立共享文件夹，共享文件夹指的是能在File station根目录中看见的。
1.点击控制面板，共享文件夹

2.新增，然后取名字比如back_up

3.指定权限为读写
这时候在你的File station根目录中就可以看见back_up了。

QuickConnect

可以用QuickConnect功能通过外网链接群辉。我试了一下外部访问那个功能，没有配成功，可能由于DDNS没有太理解，或者群辉里面那些DNS提供商被墙的原因吧。QuickConnect一次性就配成功了，只需要注册一个账号就可以了。

Time machine

群辉自带time machine功能，可以说是果粉不可缺少的功能了。由于图文较多，就不介绍了， 配置可参考官方教程。
http://www.synology.com/zh-tw/support/tutorials/481
大概思路就是建立个共享文件夹，然后制定给mac作为time machine的备份。
